# SJMC SMP2 Server-ResourcePack

上海交通大学 Minecraft 社原版生存服务器资源包

## 服务类别

ItemsAdder

* mcemoji
* mcicon
* smp2emoji -smp2自定义表情
* smp2items -smp2自定义物品
  * 更多的鱼
  * 传送卷轴 -[前置skript脚本]
* smp2title -smp2称号贴图
  * 暂无
* twitteremojis

### 注意事项
* 测试开放IA需要勤备份
* `ItemsAdder\output` 下存放的是 `/iazip` 后得到的资源包，需要解压替换
* `plugins\ItemsAdder\config.yml"` 中的 `protect-file-from-unzip` 需要全部调整为 `false` 否则将无法解压得到的资源包
